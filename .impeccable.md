# Impeccable — project design memory

## Design Context

### Users
Developers evaluating or running IDC inside Claude Code — technical readers who skim first,
then go deep. They meet the project on GitHub (README) and, now, a standalone landing page.
Context of use: assessing whether to adopt a governed Think → Plan → Build pipeline.

### Brand Personality
Three words: **systematic, confident, editorial.** It should read like a sharp consulting-forum
essay about a serious tool — calm authority, not startup hype. Emotional goal: trust + clarity.

### Aesthetic Direction
**Editorial / magazine**, cloned from the *Oliver Wyman Forum* visual language
(ref: oliverwymanforum.com). Extracted system:

- **Theme:** light. Off-white / white ground, charcoal ink, with a dark charcoal hero band for contrast.
- **Palette (OKLCH):**
  - Ink: `--ink: oklch(0.22 0.004 110)` (≈ #1a1a1a) · `--ink-2: oklch(0.33 0.003 110)` (≈ #303030)
  - Ground: `--paper: oklch(0.99 0.006 110)` (≈ #fdfdf8) · `--paper-2: oklch(0.97 0.03 116)` (≈ #f9fbe5, pale chartreuse)
  - Accent (the one electric mark): `--acid: oklch(0.86 0.19 118)` (≈ #c8dc00). Tints reduce chroma.
  - Greys for secondary text/borders: `oklch(0.6 0.005 110)` family.
- **Type:** serif display + clean grotesque body (OWF uses Tiempos + Inter; we substitute non-banned
  faces): **Source Serif 4** (display/headlines) + **Hanken Grotesk** (body/UI/labels). No Inter.
- **Scale:** big editorial display (clamp up to ~5–6rem) → 16px body. ≥1.25 ratio, fluid clamp on headings.
- **Line-height:** 1.3 headings, 1.5–1.6 body. Slightly negative tracking on display.
- **Layout:** narrow reading column (~72ch / 785px) inside wider 1150–1400px frames; generous air;
  sentence case (not all-caps); 8–10px radii; round chartreuse dots as accent marks.
- **Diagrams:** charcoal/white line-art with **chartreuse as the stroke/accent** (replacing the old
  coral/sage). The water-rig hero recolors into this language.

### Design Principles
1. **Editorial restraint.** Big serif headlines, narrow columns, lots of white space. Let type carry it.
2. **One electric accent.** Chartreuse is ~10% by visual weight — links, rules, key marks, diagram strokes.
   Charcoal + paper do the rest. Overusing the accent kills it.
3. **Serif display + grotesque body.** Source Serif 4 + Hanken Grotesk. Never Inter / system defaults.
4. **Light, confident ground.** Paper + charcoal ink; a single dark charcoal band for the hero.
5. **Show the system.** The rig diagram is the signature image — charcoal line-art, chartreuse accents.

### Anti-references (do NOT)
AI-slop tells: cyan-on-dark, purple→blue gradients, glassmorphism, gradient text, `border-left` accent
stripes, identical icon-card grids, big rounded icons above every heading, everything centered.
