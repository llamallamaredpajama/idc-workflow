`/idc:think` is a free-thinking brainstorm/interview in the **main session with zero
durable workers**: you talk, the session researches on demand via bounded fan-out, and the
output is one function-first **consideration file** under `docs/considerations/` describing
what the code should do for the user and how it behaves. No gates, no teammates, no
admission language — thinking stays free; the one PRD gate lives in Plan (`WORKFLOW.md §2`,
`§4.1`).
